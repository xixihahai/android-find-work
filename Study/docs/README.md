# Android 高级开发面试知识库

本知识库旨在帮助准备 Android 高级开发岗位面试的候选人系统性地掌握所需知识点。

## 目录结构

```
docs/
├── 01-java-kotlin-basics/          # Java/Kotlin 基础
├── 02-android-fundamentals/        # Android 基础组件
├── 03-ui-framework/                # UI 框架与渲染
├── 04-architecture/                # 架构设计
├── 05-performance-optimization/    # 性能优化
├── 06-memory-management/           # 内存管理
├── 07-power-optimization/          # 功耗优化
├── 08-network-storage/             # 网络与存储
├── 09-concurrent-programming/      # 并发编程
├── 10-framework-source/            # Framework 源码
├── 11-third-party-libs/            # 主流三方库原理
├── 12-new-technologies/            # 新技术趋势
├── 13-project-experience/          # 项目经验与场景题
└── chat-record.md                  # 聊天记录
```

## 使用说明

每个目录下包含该主题的详细知识点，包括：
- 核心概念与原理
- 常见面试题
- 实战场景分析
- 源码解析（如适用）
